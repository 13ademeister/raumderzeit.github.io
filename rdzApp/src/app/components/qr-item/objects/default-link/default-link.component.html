Dies ist dein ganz persönlicher Kronkorken. Das bedeutet du kannst alles was dir lieb ist in diesen "hineinlegen". Das funktioniert folgender Maßen:
1. Schreib uns eine E-Mail an kontakt@raum-der-zeit.de mit dem Betreff "Cup Cap Link: {{ cupId }} "
2. Gib uns Informationen, welche du hinterlegen möchtest. Zum Beispiel Text / Bild als Datei / Web-Link / Youtube-Video-Link / Musik als Datei
3. Wir kümmern uns dann darum, dass dein Kontent aus der E-Mail bei diesem QR-Code hinterlegt wird.

Sobald dein QR-Code nun gescannt wird, werden deine Informationen angezeigt!
Hier solltest du wieder auf unsere Hauptseite kommen.
